# MacBook - Support et FAQ

## Contexte
- Tous les utilisateurs utilisent **uniquement des MacBook Pro**
- Les MacBook sont gérés par **Jamf (JAMF Pro)**
- Les utilisateurs **ne sont pas administrateurs** de leur MacBook
- Les paramètres système sont gérés par Jamf via profils

## FAQ

### Installation d'applications

**Q: Comment puis-je installer de nouvelles applications sur mon MacBook?**
R: Si vous n'avez pas les droits d'administrateur, vous pouvez installer des applications depuis l'App Store en utilisant votre identifiant Apple. Si l'application requiert des permissions d'administrateur pour l'installation, veuillez contacter le support informatique pour obtenir de l'aide.

### Connexion WiFi

**Q: Comment puis-je connecter mon MacBook à un réseau Wi-Fi?**
R: Pour connecter votre MacBook à un réseau Wi-Fi, cliquez sur l'icône Wi-Fi dans la barre de menus en haut à droite de l'écran. Sélectionnez le réseau Wi-Fi auquel vous souhaitez vous connecter et entrez le mot de passe si nécessaire.

**Note importante**: Le réseau WiFi du bureau est uniquement pour les MacBook et est géré par des profils Jamf. Si vous avez des problèmes de connexion, consultez la documentation WiFi spécifique.

### Captures d'écran

**Q: Comment prendre une capture d'écran sur mon MacBook?**
R: Pour prendre une capture d'écran sur votre MacBook, appuyez sur 'Command + Shift + 3' pour capturer l'intégralité de l'écran. Pour capturer une partie spécifique de l'écran, utilisez 'Command + Shift + 4', puis sélectionnez la zone que vous souhaitez capturer.

### Mode sombre

**Q: Comment puis-je configurer le mode sombre sur mon MacBook?**
R: Pour activer le mode sombre, allez dans 'Préférences Système', puis cliquez sur 'Général'. Sous 'Apparence', sélectionnez 'Sombre'. Si vous ne pouvez pas accéder à cette option, veuillez contacter le support informatique.

### Mises à jour macOS

**Q: Comment puis-je mettre à jour macOS?**
R: Si vous n'avez pas les droits d'administrateur, vous ne pourrez pas mettre à jour macOS vous-même. Veuillez contacter le support informatique pour effectuer la mise à jour de votre système d'exploitation.

### Espace de stockage

**Q: Comment libérer de l'espace de stockage sur mon MacBook?**
R: Vous pouvez libérer de l'espace en supprimant des fichiers inutiles ou en transférant des fichiers vers un stockage externe ou iCloud. Pour voir ce qui occupe de l'espace, allez dans 'À propos de ce Mac' > 'Stockage' > 'Gérer'.

### Personnalisation du Dock

**Q: Comment puis-je personnaliser le dock sur mon MacBook?**
R: Pour personnaliser le dock, allez dans 'Préférences Système', puis cliquez sur 'Dock & Barre de menus'. Vous pouvez y modifier la taille du dock, sa position sur l'écran, et d'autres options.

### Imprimantes

**Q: Comment puis-je ajouter une imprimante à mon MacBook?**
R: Allez dans 'Préférences Système', puis 'Imprimantes et Scanners'. Cliquez sur le bouton '+' pour ajouter une nouvelle imprimante. Si vous avez besoin d'une permission administrateur pour cette opération, veuillez contacter le support informatique.

### Sauvegarde iCloud

**Q: Comment puis-je sauvegarder mes fichiers sur iCloud?**
R: Pour sauvegarder vos fichiers sur iCloud, ouvrez 'Préférences Système', puis cliquez sur 'Identifiant Apple' et 'iCloud'. Activez iCloud Drive et sélectionnez les dossiers que vous souhaitez sauvegarder. Assurez-vous d'avoir suffisamment d'espace sur iCloud.

### Performance

**Q: Que faire si mon MacBook est lent ou ne répond pas?**
R: Vous pouvez essayer de redémarrer votre MacBook pour résoudre les problèmes de lenteur. Si le problème persiste, fermez les applications qui consomment beaucoup de ressources via le 'Moniteur d'activité'. Si cela ne fonctionne pas, contactez le support informatique pour un diagnostic plus approfondi.

### Trackpad

**Q: Comment puis-je ajuster la sensibilité du trackpad de mon MacBook?**
R: Pour ajuster la sensibilité du trackpad, allez dans 'Préférences Système', puis sélectionnez 'Trackpad'. Vous y trouverez des options pour ajuster la vitesse du curseur, les gestes, et la sensibilité du clic.

**Q: Comment désactiver le son du clic sur le trackpad?**
R: Pour désactiver le son du clic sur le trackpad, allez dans 'Préférences Système', puis 'Trackpad'. Décochez l'option 'Clic silencieux' sous l'onglet 'Point & Clic'.

### Nettoyage

**Q: Comment nettoyer l'écran et le clavier de mon MacBook?**
R: Pour nettoyer l'écran, utilisez un chiffon doux et non pelucheux légèrement humide. Pour le clavier, vous pouvez utiliser une lingette légèrement humide ou un chiffon en microfibre. Assurez-vous que votre MacBook est éteint et déconnecté de l'alimentation avant de le nettoyer.

### Clavier

**Q: Mon clavier MacBook ne répond pas correctement, que puis-je faire?**
R: Si certaines touches du clavier ne répondent pas, assurez-vous que rien n'obstrue les touches. Vous pouvez essayer de souffler de l'air comprimé entre les touches pour éliminer la poussière. Si le problème persiste, contactez le support informatique.

**Q: Puis-je remplacer le clavier rétroéclairé de mon MacBook par moi-même?**
R: Le remplacement du clavier rétroéclairé nécessite des compétences techniques avancées et l'accès administrateur. Il est recommandé de contacter le support informatique pour toute réparation ou remplacement matériel.
