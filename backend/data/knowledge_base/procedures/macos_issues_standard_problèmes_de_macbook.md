# Problèmes de macbook

**Catégorie:** macos_issues
**Description:** Procédure standard pour problèmes de macbook

## Questions de diagnostic

- Quel est le problème exact ?
- Depuis quand le problème existe-t-il ?

## Étapes de résolution

### Étape 1: Premier diagnostic pour identifier si c’est un problème matériel ou logiciel ou infrastructure (Réseau, 2eme écran etc)

### Étape 2: Procédures standards de résolution de problèmes N1

### Étape 3: Si non résolu, ne pas insister et envoyer un ticket vers odoo avec les détails et les étapes déjà fait


## Création de ticket Odoo

**Quand créer un ticket:** Si le problème n'est pas résolu après les tentatives de diagnostic

**Champs requis:**
- **title:** Résumé de la demande
- **description:** Tous les détails collectés, étapes déjà effectuées, et informations de diagnostic
- **priority:** Normal
