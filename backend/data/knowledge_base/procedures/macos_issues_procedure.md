# Procédure de Support pour Problèmes macOS

**Catégorie:** macos_issues
**Description:** Procédure standardisée pour diagnostiquer et résoudre les problèmes courants sur macOS.

## Questions de diagnostic

- Quel est le modèle et l'année de votre Mac?
- Avez-vous récemment installé une mise à jour ou un nouveau logiciel?
- Pouvez-vous décrire le problème en détail?
- Avez-vous redémarré votre Mac depuis l'apparition du problème?
- Le problème est-il lié à une application spécifique?

## Étapes de résolution

### Étape 1: Vérifier les mises à jour macOS

Accédez à Préférences Système > Mise à jour de logiciels et installez les mises à jour disponibles.

### Étape 2: Redémarrer le Mac

Demandez à l'utilisateur de redémarrer le Mac pour voir si le problème persiste.

### Étape 3: Réinitialiser la NVRAM

Éteignez le Mac, puis rallumez-le en maintenant les touches Option + Command + P + R pendant environ 20 secondes.

### Étape 4: Vérifier les autorisations des fichiers (nécessite droits admin)

Utilisez l'Utilitaire de disque pour réparer les autorisations des fichiers.

### Étape 5: Réinstaller l'application problématique

Si le problème est lié à une application spécifique, désinstallez et réinstallez l'application.


## Création de ticket Odoo

**Quand créer un ticket:** Si le problème persiste après avoir suivi toutes les étapes de résolution

**Champs requis:**
- **title:** Problème macOS - [Description courte du problème]
- **description:** Inclure le modèle du Mac, la version de macOS, les étapes déjà suivies et les résultats obtenus.
- **priority:** Moyenne

## Problèmes fréquents

- Problèmes de mise à jour macOS
- Problèmes de connexion réseau
- Applications qui ne répondent pas
- Problèmes de démarrage
